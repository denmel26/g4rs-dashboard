Power BI Dashboard Integration

To integrate your Power BI report, follow these steps:

Method 1: Public Report (Publish to Web)

  1. In Power BI Service, open your report
  2. Go to File > Embed report > Publish to web
  3. Click Create embed code
  4. Copy the iframe code and replace the commented iframe above

Method 2: Secure Embed (Private Report)

  1. In Power BI Service, go to File > Embed report > Website or portal
  2. Copy the secure embed code
  3. This requires additional authentication setup

Method 3: Direct Link

  1. Get your Power BI report URL
  2. Use it as the iframe src (may require authentication)
